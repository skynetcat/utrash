# Features
1. Small.
2. Readability.
5. Uses fzf.
3. Has what's expected: 
    - Trash files.
    - Erase files from trash. 
    - Empty the trash.
    - Recover files from trash.
    - List trashed files.
4. Date filters:
    - Today.
    - Yesterday.
    - This month.
    - This year.

# Installation steps
## 1. Installing required dependencies
### 1.1 Termux
```bash
pkg install git fzf
```
## 2. Cloning the repository
```bash
git clone https://github.com/skynetcat/utrash.git
cd utrash
```
## 3. Run the installer
```bash
chmod +x install 
./install
```
